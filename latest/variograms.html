Variograms

Variograms

\[\newcommand{\x}{\boldsymbol{x}} \newcommand{\1}{\mathbb{1}}\]

In an intrinsically stationary isotropic model, the variogram is only a function of the distance between any two points $\x_1,\x_2 \in \mathbb{R}^m$:

\[\gamma(\x_1,\x_2) = \gamma(||\x_1 - \x_2||) = \gamma(h)\]

The same holds for the covariance, which is directly related $\gamma(h) = cov(0) - cov(h)$. This package implements a few commonly used stationary models:

Gaussian

\[\gamma(h) = (s - n) \left[1 - \exp\left(-\left(\frac{h}{r}\right)^2\right)\right] + n \cdot \1_{(0,\infty)}(h)\]

  • r ∈ ℜ - range

\[\gamma(h) = (s - n) \left[\left(\frac{3}{2}\left(\frac{h}{r}\right) + \frac{1}{2}\left(\frac{h}{r}\right)^3\right) \cdot \1_{(0,r)}(h) + \1_{[r,\infty)}(h)\right] + n \cdot \1_{(0,\infty)}(h)\]
